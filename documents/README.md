Scripts, summaries and notes from my studies. Browse by school below, then by
course.

**How to add material (all from the browser):** open the folder for a course on
GitHub, click **Add file → Upload files**, drag your PDFs / code / notes in, and
commit. They appear here automatically with a viewer and a download link.

To add a **new school or course**, just create a new folder (commit any file
inside it). Drop a `README.md` with a `title:` into a folder to give it a nicer
display name and a short description.
